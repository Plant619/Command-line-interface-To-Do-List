# Command-line-interface-To-Do-List
The application stores tasks as JavaScript objects and allows users to interact with them through a console menu. Users can repeatedly select options until exiting the program

# Features
Display Tasks

Displays all stored tasks

Shows completion status using:
[✓] Completed
[ ] Incomplete

Add Tasks
- Allows users to add new tasks dynamically
-- Newly added tasks are automatically marked as incomplete

Delete Tasks
- Removes selected tasks from the task list

Mark Task as Complete
- Updates selected tasks as completed

Mark Task as Incomplete
- Allows completed tasks to be changed back to incomplete

# Technologies Used
- Node.js
- JavaScript
- readline-sync

# Setup 
1. Clone repository
2. npm install
3. Run program (Type node list.js)
